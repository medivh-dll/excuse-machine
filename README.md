# 🤥 Excuse Machine

> *Certified Plausible Deniability Generator*

A brutalist, black-and-white excuse generator that produces handcrafted, suspiciously believable excuses for every situation. Built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies, no regrets.

---

## 🚀 Live Demo

[**Try it here →**](https://medivh-dll.github.io/excuse-machine)

---

## ✨ Features

- **6 Categories** — Work, School, Social, Family, Health, and Pure Chaos
- **30+ handcrafted excuses** with deadpan attribution labels
- **Typewriter animation** — every excuse types itself out character by character
- **Believability Rating** — rate each excuse from *BUSTED* to *ACADEMY AWARD*
- **One-click Copy** — grab any excuse instantly for deployment
- **Scrolling ticker** — ambient excuse headlines in the background
- **Zero dependencies** — pure HTML/CSS/JS, works offline

---

## 🎨 Design

Brutalist black-and-white aesthetic. Monospace/display font pairing (VT323 + Anton + Oswald). Inspired by terminal interfaces and old-school newsprint. Part of a series of single-page generator tools.

---

## 🛠️ Usage

Just open `index.html` in any browser. No build step, no install, no npm.

```bash
git clone https://github.com/medivh-dll/excuse-machine.git
cd excuse-machine
open index.html
```

Or deploy instantly with **GitHub Pages**:

1. Go to repo **Settings → Pages**
2. Set source to `main` branch, `/ (root)`
3. Done — it's live
